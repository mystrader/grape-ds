# Button

With `MDX` we can define a story for `Button` right in the middle of our
markdown documentation.

### Primary button

<Button
  color="primary"
  variant="standard"
  size="regular"
  disabled={false}
  onClick={() => { console.log('Button click') }}
>
  Primary
</Button>
